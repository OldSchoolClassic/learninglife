# Learning Life

Learning Life is a polished, multi-page website for Alona Cortes and her homeschool support services in Pennsylvania. The site presents homeschool evaluations, IEP support, graduation and diploma guidance, tutoring, starting-homeschooling help, and curated curriculum/resources in a clean, premium, family-centered format.

The site is built as a static GitHub Pages site with Jekyll processing enabled for clean URLs.

## Live Site

- Custom domain: `https://learninglife.us`

## Core Features

- Clean, professional public-facing website
- Premium responsive UI for desktop, tablet, and mobile
- Jekyll-powered clean URLs such as `/about/` instead of `/about.html`
- Centralized CSS and JavaScript
- Multi-page structure with consistent navigation, footer, and design system
- Homeschool-specific service pages and resources tailored to Pennsylvania families
