# Pillars of Peace Primary Partners

Static Jekyll site for Alona Cortes — PA homeschool evaluator.

## Pages
- `index.html` — Home
- `about.html` — About Alona
- `evaluations.html` — Preparing for Your Evaluation
- `iep.html` — Students with IEPs
- `graduation.html` — Graduation & Diplomas
- `start-homeschooling.html` — Start of Year Forms
- `resources.html` — Curriculum & Resources
- `faqs.html` — Frequently Asked Questions
- `schedule.html` — Schedule an Evaluation
- `contact.html` — Contact

## Deploy
Push to the `main` branch of your GitHub repository. Enable GitHub Pages in Settings > Pages > Source: Deploy from branch `main`.

## Local Development
```bash
bundle install
bundle exec jekyll serve
```
